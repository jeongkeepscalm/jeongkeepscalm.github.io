## REGEXP ( LIKE 여러 단어 )

```sql
SELECT
    animal_id
    , name
    , case 
        when sex_upon_intake regexp 'Neutered|Spayed'
        then 'O'
        else 'X'
    end as sex_upon_intake
from animal_ins
order by 1 asc;
```

## 가장 비싼 가격 정보 출력
```sql
select 
    product_id
    , product_name
    , product_cd
    , category
    , price
from food_product
where price = (select max(price) from food_product);
```
