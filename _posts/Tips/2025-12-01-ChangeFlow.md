## 💡 체인지플로우 커밋 및 업데이트 방법

- 신규파일 커밋
  - 신규파일 → 체크인(파일 드래그) → 웹 접속 → 필드 및 결과 → 배포

- 업데이트
  - 내 작업목록 → 행 더블클릭 → 자원요청 탭에 파일 드래그 → 웹 접속 → 반출 요청 → 반출할 파일 선택 후 반출 → 이클립스 리소스 목록 → 소스 선택 → 우클릭 → 체크아웃 → 추후 병합 → 개발 체크인 → 빌드 및 결과 → 배포

- 자원요청해제: 리소스 목록 삭제
